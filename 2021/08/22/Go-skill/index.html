1.值类型和引用类型

值类型与引用类型

  • 值类型：变量直接存储值，内存通常在栈中分配（属于值类型的数据类型有：int、float、bool、string、数组以及struct）。 零值对应的是对应数据类型的零值。

  • 引用类型：变量存储的是一个地址，这个地址存储最终的值，内存通常在堆中分配，通过GC回收（属于引用类型的数据类型有：指针、slice、map、chan等）。 零值对应的是nil.

  • 注意:

    • 值类型当做参数传递，是copy传递，copy一份过去，改的是新的，老的不会受影响

    • 其实各个标准资料中没有说明Python有值类型和引用类型的分类。但是可以理解为Python中没有值类型，全是引用类型。Python 的所有变量其实都是指向内存中的对象的一个指针，所有的变量都是！

      • 对象还分两类：（有可变类型和不可变类型之分，可变类型改的是原来的，不可变的是新复制的一份传过去，不改变原来的）可修改(mutable)的类型叫做值类型，不可修改(immutable)类型叫做引用类型

    • 引用类型当做参数传递，复制地址传过去，会影响原来的

4.拷贝

有时候 为了不让对象直接相互干扰, 我们需要深度赋值对象

浅拷贝

引用类型的数据，默认都是浅拷贝

  • slice map funtion
  • 新老对象指向同一块内存区域
  • a修改字段，b中字段也跟着改变了
  • 内存销毁是一致的

深拷贝

值类型的数据，默认都是深拷贝

  • int float string bool array struct

  • 复制的时候，会创建一个对象

  • 指向完全不同的内存地址

  • 修改是互不影响的
