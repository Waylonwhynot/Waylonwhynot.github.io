1.3.2 单线程为什么这么快

1 纯内存

2 非阻塞IO （epoll），自身实现了事件处理，不在网络io上浪费过多时间

3 避免线程间切换和竞态消耗

1.3.3 注意

1 一次只运行一条命令

2 拒绝长慢命令

-keys，flushall,flushdb,慢的lua脚本，mutil/exec，operate，big value

3 其实不是单线程（在做持久化是另外的线程）

-fysnc file descriptor

-close file descriptor
