前端-VUE-14-Vue 组件化注意事项

Vue 组件化注意事项

  • 组件可以理解为特殊的 Vue 实例，管理自己的 template 模板。
  • 组建的 template 必须有且只有一个根节点。
  • 组件的 data 选项必须是一个函数，且函数体返回一个对象。
  • 组件与组件之间是相互独立的，可以配置自己的一些选项资源 data 、 methods 、 computed 等等。
  • 思想：组件自己管理自己，不影响别人。
