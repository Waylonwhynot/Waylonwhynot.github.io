VUE 事件修饰符以及按键码

之前写了一篇关于事件处理的文章，这篇的时间修饰符主要就是来修饰之前事件处理的事件名的，目的是为了扩展事件的功能。

事件修饰符

  • .stop：阻止单击事件继续传播，event.stopPropagation()
  • .prevent：阻止事件默认行为 event.preventDefault()
  • .once：点击事件将只会触发一次
