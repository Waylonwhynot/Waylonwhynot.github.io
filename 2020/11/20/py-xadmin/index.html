Django-xadmin的介绍

Django是python的重量级web框架,写得少,做得多,非常适合后端开发,它很大的一个亮点是,自带后台管理模块,但它自带的后台管理有点丑,而Xadmin是基于bootstrap开发的一套后台管理框架,界面非常美观,只需几步就可以替换自带的Django_admin

具体的安装步骤
  1. xadmin在python2.x时代的安装方法

在python2.x时代，安装xadmin是通过如下命令

pip install xadmin
